# ttTv

去广告视频播放器

## 主要功能
1. 搜索全网在线播放视频
2. 搜索可迅雷下载资源
3. 可播放会员视频
3. 跳过广告


## 目录
1. backend 是后台程序，提供搜索影视资源接口
2. client 是桌面客户端程序
3. 移动端客户端程序（暂时不可用）

##使用方法

```
    cd backend/node
    node service.js
    
    cd client
    yarn install #或 npm install
    yarn dev #请勿关闭进程
    yarn electron #新开终端
```
